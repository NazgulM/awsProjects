# Group 1 Project

```
Group1: Create resources and perform actions through AWS Console
1. Create 3 images with apache web server installed. 1st image should contain "version 1" in index.html, 2nd image should contain "version 2" in index.html and 3rd image should contain "canary" in index.html.
2. Create 2 VMs from each image (total 6 VMs)
3. Create load balancer for 6 VMs (create 3 target groups for each image)
4. Assign your DNS that it will send traffic to your load balancer
5. Perform blue-green deployment
6. Perform canary deployment
```
